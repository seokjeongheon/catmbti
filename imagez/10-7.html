웹 페이지란?

웹 페이지는 웹 서핑을 할 때 보는 각각의 화면을 말한다. 웹 페이지는 html 과 css 로 구성된 html문서와 관련된 이미지, 동영상, 음악 파일 등의 데이터 파일로 구성된다.
